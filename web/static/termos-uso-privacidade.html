Termos de Uso do Aplicativo Votei

1. Aceitação dos Termos

Ao acessar ou usar o aplicativo Votei (doravante denominado "Aplicativo"), você concorda em cumprir e estar vinculado aos seguintes Termos de Uso. Caso não concorde com estes termos, você não deve utilizar o Aplicativo.

2. Coleta e Uso de Dados

2.1 Coleta e Uso de Dados

Para fornecer e melhorar nossos serviços, coletamos as seguintes informações:

  • Dados de Cadastro: Nome, e-mail, foto de perfil (se fornecida).
  • Dados de Acesso: Dados de login do Google ou Facebook, incluindo informações associadas a essas contas.
  • Dados de Localização: Localização geográfica para validação dos votos.
  • Dados de Dispositivo: IP, MAC e informações sobre o dispositivo utilizado.

2.2 Finalidade da Coleta

Os dados coletados têm as seguintes finalidades:

  • Validação dos Votos: Os dados são usados para garantir a integridade dos votos e prevenir fraudes, como a votação múltipla por uma única pessoa.
  • Segurança e Integridade: Os dados de localização, IP e MAC são utilizados para verificar a autenticidade dos votos e manter registros de logs para fins de segurança.
  • Melhoria dos Serviços: Utilizamos as informações para melhorar a experiência do usuário e otimizar o funcionamento do Aplicativo.

2.3 Armazenamento e Segurança

Seus dados são armazenados com segurança e protegidos contra acesso não autorizado. No entanto, não podemos garantir a segurança absoluta das informações transmitidas pela internet.

3. Modos de Acesso

3.1 Login Social

Ao fazer login com Google ou Facebook, você nos fornece dados de sua conta social para fins de autenticação e validação.

3.2 Acesso Sem Login

Se optar por acessar o Aplicativo sem login, seu voto terá um peso reduzido em comparação aos votos realizados com login. Dados básicos como localização e IP ainda serão coletados para validação.

4. Modificações nos Termos

Podemos atualizar estes Termos de Uso periodicamente. A sua continuidade no uso do Aplicativo após tais modificações constituirá sua aceitação das novas condições.

5. Contato

Se você tiver dúvidas sobre estes Termos de Uso ou sobre como lidamos com seus dados, entre em contato conosco pelo e-mail: votei.contato@proton.me

6. Disposições Gerais

Estes Termos de Uso são regidos pelas leis [do Brasil/estado Bahia]. Qualquer disputa relacionada a estes termos será resolvida nos tribunais competentes.
